## 正则表达式

由原子，元字符，以及模式修正字符组成。**一个正则表达式中至少包含一个原子**

* ### 匹配单个字符

  | 字符 | 功能                                |
  | ---- | ----------------------------------- |
  | .    | 匹配任意一个字符                    |
  | []   | 匹配[]中列举的字符                  |
  | \d   | 匹配数字，即0~9                     |
  | \D   | 匹配非数字，即不是数字              |
  | \s   | 匹配空白，即空格，\t~tab键，\n~换行 |
  | \S   | 匹配非空白                          |
  | \w   | 匹配单词字符，即a~z、A~Z、0~9、_    |
  | \W   | 匹配非单词字符                      |
